Know how to type in one language?

Bukva lets you type in any language.

What is it?

Bukva is a tool for instantaneous transliteration between any two languages anywhere on Windows. Start typing right away in Russian, Greek, Kyrgyz, or Uzbek.

No need for copy and pasting! The translation happens as you type before the letters even appear on screen.

Supported Languages

Bukva currently supports the following language transliterations:

  • English to Russian
  • English to Greek
  • English to Uzbek
  • English to Kyrgyz

Downloads

To run, download any language option, extract the zip file, and open Bukva.exe.
