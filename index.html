Know how to type in one language?

Bukva lets you type in any language.

What is it?

Bukva is a tool for instantaneous transliteration anywhere on Windows.

Supported Languages

Bukva currently supports the following mappings:

  • English → Russian
  • English → Greek
  • English → Uzbek
  • English → Kyrgyz

Downloads

To run, download any language option, extract the zip file, and open Bukva.exe.
